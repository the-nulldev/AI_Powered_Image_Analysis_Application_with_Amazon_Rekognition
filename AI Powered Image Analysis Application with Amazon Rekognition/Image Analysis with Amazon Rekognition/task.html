Theory

Our application needs to automatically analyze the images stored in S3 to ensure that only high-quality pictures of the right types of produce make it through. To achieve this, we will be leveraging Amazon Rekognition. This is an image and video analysis service provided by AWS. It uses deep learning models to identify objects, people, text, scenes, and activities in images and videos, as well as to detect any inappropriate content. Let's discuss some key features and capabilities of Amazon Rekognition.

Label detection: This feature identifies objects, people, text, scenes, and activities in images. For example, it can detect objects like cars, trees, and furniture, or activities like running and dancing. You can use the console, SDKs, or CLI commands to detect labels in an image. For instance, to detect images in a file cow.jpg stored in Amazon S3, you could use the following command:

# if you use PowerShell as your shell, replace backslashes (\) with backticks (`) as line seperators

aws rekognition detect-labels \
   --image '{"S3Object":{"Bucket":"your-unique-bucket-name","Name":"cow.jpg"}}' \
   --region us-east-1

Face detection and analysis: Amazon Rekognition can detect faces in images and videos and analyze attributes such as age range, gender, emotions, and more. It can also compare faces to determine if they are the same person. Again, you can use any tool you like to interact with the DetectFaces API:

# if you use PowerShell as your shell, replace backslashes (\) with backticks (`) as line seperators

aws rekognition detect-faces \
    --image '{"S3Object":{"Bucket":"your-unique-bucket-name","Name":"example.jpg"}}' \
    --attributes "ALL" \
    --region us-east-1

Face comparison: This feature allows you to compare a face in an image with faces in another image or in a collection of faces. Here's a CLI command to achieve this:

aws rekognition compare-faces \
    --source-image '{"S3Object":{"Bucket":"your-unique-bucket-name","Name":"source.jpg"}}' \
    --target-image '{"S3Object":{"Bucket":"your-unique-bucket-name","Name":"target.jpg"}}' \
    --region us-east-1

Text in image: Amazon Rekognition can detect and recognize text in images, which is useful for extracting information from documents, street signs, or product labels.

Celebrity recognition: This feature identifies celebrities in images and videos. It can recognize thousands of famous personalities from various fields. Try it out by uploading a celebrity's image to an S3 bucket and running the following CLI command:

aws rekognition recognize-celebrities \
    --image '{"S3Object":{"Bucket":"your-unique-bucket-name","Name":"celebrity.jpg"}}' \
    --region us-east-1

Custom labels: You can train Amazon Rekognition to detect custom labels specific to your business needs, such as identifying machine parts or analyzing medical images. This requires you to train a model that your applications will afterwards use. 

Video analysis: Amazon Rekognition can analyze videos to detect activities, people, objects, and more. It can also track the movement of objects and people across frames. This is important for surveillance systems. 

Content moderation: This feature helps to detect inappropriate content in images and videos, such as nudity or violence, making it easier to moderate user-generated content.

The response from Amazon Rekognition includes a list of detected labels, each with a name, confidence score, and additional details. The confidence score is a measure of how certain the model is about the label it has detected, expressed as a percentage. Higher confidence scores indicate greater certainty. For example, a confidence score of 98.5% for the label "Car" means the model is very certain that the object in the image is a car.
